The SNU Browser Nose - High level browser sniffing¶

This page is derived from SourceBase and may contain some template elements. Make sure to point them out when found and use the latest version of the page.

About

The SNU Browser Nose is a database of browser sniffing scripts with the goal of making SNU work with every browser.

For security reasons, all browser versions before 2014 are blacklisted outside of VM usage due to security issues, however permissions will be available upon choice.

Notice about universal browser sniffing

I am aware that browser sniffing may not be possible in some versions of each browser (examples: Mosaic, ViolaWWW, etc) or even older browsers due to the invention and use of user agent strings. I will try to use other methods, and fallback basic HTML methods
